# Qexo
一个快速、强大、漂亮的在线 Hexo 编辑器，您的 Star 是对我最大的支持。 [Wiki](https://github.com/am-abudu/Qexo/wiki)
![](https://user-images.githubusercontent.com/51912589/159258766-19a1ce22-d34b-4b29-b291-7d70e8942859.png)
## 特色功能
- 自定义图床上传图片
- 在线配置编辑
- 在线页面管理
- 开放 API
- 自动检查更新
- 实验性的在线更新
- 自动填充 date 模板
- 基于时间戳的 abbrlink
